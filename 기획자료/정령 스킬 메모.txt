정령 스킬 :: 
기본적으로 사용하고 있는 내용의 리스트,

인덱스: 말그대로 인덱스
위력: 사용하는 스킬의 데미지 / 회복 수치, 앞선 감지 대상이 아군이면 위력에 적혀있는 수치만큼 힐을 한다.
범위: 앞의 Usetype에 따라 1이면 타일 단위, 2면 M단위로 계산
쿨타임 : 정령의 소환 가능한 쿨타임 -> 이걸 여기서 처리해서 쫄이 사용할 수 있게 하자.
형태: 근접, 범위, 단일, 방출의 형태 확인 -> 피격은 에셋이 가지고 있는 충돌 체크로 가능한지 프로그래머랑 확인 해야 됨
	근접 : 피격 이펙트만 터지고 투사체는 없음 앞선 감지 대상을 직접 공격
	광역 : 바라보는 방향을 중심으로 퍼지는 범위 
	단일 : 대상을 향해 투사체 발사 -> 유도 성능을 가진다. 
	방출 : 방향을 향해 발사 -> 유도 성능이 없다. 

이동 속도 : 투사체가 날아가는 속도 
대상: 피해를 입히는 대상의 숫자, 여기에 뭐가 들어가냐에따라 능력이 발동 했을 때 피격 되는 대상을 측정
생존: 쉽게 말해 지속시간, 살아 있는 시간 동안 단일을 제외한 모든 투사체가 지속 시간동안 주기 마다 피격 판정을 리셋 한다.
주기 : 장판형의 피격 주기 및, 근접과 단일 스킬의 발사 주기
버프 : 사용되는 버프 -> 위력에 상관 없이 해당 능력에 피해를 받으면 버프가 적용된다.
추가 : 사용 이후 생성 될 수 있는 인덱스. 생성 지점은 투사체가 종료된 지점을 0, 0으로 지정 한다.
발사 프리팹 : 발사시 이팩트가 생성되는 위치
투사체 프리팹 : 발사되는 투사체
피격 프리팹 : 피격된 대상에 생성되는 이펙트
